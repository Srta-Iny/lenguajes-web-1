HTML+CSS+Javascript

  • HTML 5

    HTML5 (HiperText Markup Languaje, version 5) es la quinta revisión immportante del lenguaje básico de la World Wide Web.

    Usa la extencion .html

  • CSS 3

    Hoja de estilo en cascada o CSS(siglas en ingles de cascading style sheets) es un lenguaje usado para definir y crear la presentacion de un documento estructurado escrito en HTML.

    Usa la extención css

  • JavaScript

    JavaScript(abreviado comunmente "JS") es un lenguaje de programación interpretado utilizando principalmente en el lado del cliente(client-sided), imprementado como parte de un nevegador web permitiendo mejoras en la interfaz de usuario y paginas web ounamicas.

    Usa la extención .js
